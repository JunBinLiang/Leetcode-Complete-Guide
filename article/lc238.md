### :bulb:代码：
```
    public int maxProfit(int[] prices) {
        int n = prices.length;
        int maxSell = prices[n - 1];
        int maxProfit = 0;

        for (int i = n - 2; i >= 0; i--) {
            maxProfit = Math.max(maxProfit, maxSell - prices[i]);
            maxSell = Math.max(maxSell, prices[i]);
        }
        return maxProfit;
    }
```

#### 空间复杂度和时间复杂度：
  - 时间复杂度：O(N) 
  - 空间复杂度：O(1)
