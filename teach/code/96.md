# Unique Binary Search Trees


## Java

```
class Solution {
    int dp[][];
    public int numTrees(int n) {
        dp=new int[n+1][n+1];
        
        for(int i=0;i<dp.length;i++){
            Arrays.fill(dp[i],-1);
        }
        
        return dfs(1,n);
    }
    
    public int dfs(int l,int r){
        if(l>=r){
            return 1;
        }

        if(dp[l][r]!=-1)return dp[l][r];
        int res=0;
        for(int i=l;i<=r;i++){
            res+=dfs(l,i-1)*dfs(i+1,r);
        }
        dp[l][r]=res;
        return res;
    }
}
```
